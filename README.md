# Project Name

Tourist Bureau Management System (C++)

# Description

Данный проект представляет собой консольное приложение на языке C++, предназначенное для управления туристическим бюро. Программа обеспечивает аутентификацию и авторизацию пользователей с различными ролями (водитель, гид, администратор, менеджер). Система взаимодействует с базой данных SQLite и хранит информацию об автобусах, экипажах, маршрутах и выполненных рейсах. Реализованы основные операции работы с данными (просмотр, добавление, обновление и удаление), а также расчёт стоимости поездок и доходов экипажей. Дополнительно проект покрыт автоматическими тестами с использованием Google Test.

# Installation

1. Клонировать репозиторий:

```bash
git clone git@github.com:hhlbby/515team_code.git
cd 515team_code
```

2. Установить зависимости:

* CMake
* SQLite3
* Компилятор с поддержкой C++17

3. Собрать проект:

```bash
mkdir build
cd build
cmake ..
make
```

4. (Опционально) Запустить тесты:

```bash
cd bin
./tests
```

# Usage

1. Запустить приложение:

```bash
cd bin
./TouristBureau
```

2. Ввести логин и пароль пользователя.

3. После успешной авторизации использовать консольное меню для:

* просмотра автобусов, маршрутов и рейсов
* получения информации об экипаже
* добавления и редактирования данных
* расчёта стоимости поездок и доходов

4. Для проверки корректности работы рекомендуется:

* запускать автоматические тесты (`./tests`)
* использовать тестовую базу данных
* проверять изменения через SQLite (например, DB Browser)
